# Network Port Scanner

Ferramenta simples em Python para escanear portas abertas em um host alvo.

## Funcionalidades

- Escaneia portas de 1 a 1024
- Identifica quais portas estão abertas
- Ferramenta simples para reconhecimento de rede

## Como usar

Execute o script no terminal:

python scanner.py

Depois digite o IP ou domínio que deseja escanear.
